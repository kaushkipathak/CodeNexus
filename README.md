# CodeNexus
Gamified EdTech platform for learning DSA
A gamified EdTech platform designed for undergraduate students to learn Data Structures and Algorithms (DSA) from scratch without losing motivation. 

Code Nexus uses game design elements, interactive visual animations, and a learn-by-doing approach to make mastering DSA engaging and rewarding. Focus on your growth through skill trees, daily challenges, and a sleek dark-mode UI.

 Key Features

- **Gamified Learning**: Earn XP, climb ranks, and earn badges as you solve problems.
- **Skill Tree Progression**: Unlock chapters and new algorithms just like skills in an RPG game.
- **Daily Challenges & Streaks**: Stay consistent with daily coding challenges and a streak tracking system.
- **Interactive Visuals**: Understand complex data structures and algorithms through rich, interactive animations.
- **Tiered Difficulty**: Problems automatically scale to match your current skill level.
- **Weekly Leaderboards**: Compete with peers globally to claim top spots.
- **Modern UI**: A sleek, minimal, and modern dark-themed user interface designed for extended focus.

- Tech Stack

Code Nexus is built using modern web development technologies (MERN Stack):

**Frontend:**
- [React.js](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [React Router DOM](https://reactrouter.com/)
- [Axios](https://axios-http.com/)

**Backend:**
- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/) & [Mongoose](https://mongoosejs.com/)
- [JWT (JSON Web Tokens)](https://jwt.io/) & [Bcrypt.js](https://www.npmjs.com/package/bcryptjs)
